# Sales Analytics Dashboard

## 📊 Overview
This dashboard analyzes sales performance across regions, products, and channels.

## 🔹 Key Metrics
- Total Sales
- Average Monthly Sales
- Total Returns & Return %
- Quantity Sold
- Average Selling Price

## 🔹 Insights
- West and North regions contribute the highest sales
- Returns are concentrated in specific product categories
- Distributor channel dominates sales contribution

## 🔹 Features
- Interactive filters (Month, City, State)
- Region-wise comparison
- Product-level performance analysis

## 🛠 Tools
Power BI, Python (EDA)
