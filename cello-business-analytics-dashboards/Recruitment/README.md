# HR Recruitment Analytics Dashboard

## 📊 Overview
This dashboard provides insights into recruitment performance and hiring efficiency.

## 🔹 Key Metrics
- Total Requisitions
- Offer Acceptance Rate
- Average TAT
- Cost per Hire

## 🔹 Insights
- Majority hiring concentrated in key departments
- Offer acceptance rate indicates strong candidate conversion
- Certain roles exceed SLA timelines

## 🔹 Features
- Recruitment funnel visualization
- Department-wise hiring analysis
- SLA tracking

## 🛠 Tools
Power BI, Python
