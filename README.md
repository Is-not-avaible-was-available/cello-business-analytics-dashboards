# Cello Business Analytics Dashboards

This repository contains analytics dashboards developed during my internship at Cello World Ltd., focusing on Sales and HR analytics.

## 🔹 Tools Used
- Python (Pandas, NumPy)
- Power BI
- Tableau
- Figma

## 🔹 Dashboards Included

### 1. Sales Analytics Dashboard
- Regional performance analysis
- Product-level insights
- Returns and sales trends
- Channel and payment analysis

### 2. HR Recruitment Dashboard
- Hiring funnel analysis
- Offer acceptance rate
- Turnaround time (TAT)
- Department-wise hiring insights

## 🔹 Key Impact
- Reduced manual reporting effort by ~40%
- Improved visibility into business KPIs
- Enabled faster and data-driven decision-making

## 🔹 Note
Datasets are not fully shared due to confidentiality. Sample structures may be included.
